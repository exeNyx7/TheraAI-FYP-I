# Frontend Components

This directory contains reusable React components for the TheraAI application.

## Structure

- `Navbar.jsx` - Main navigation component
- `Footer.jsx` - Footer component
- `LoadingSpinner.jsx` - Loading indicator
- `ChatBot.jsx` - AI therapy chat interface
- `TherapySession.jsx` - Therapy session component
- `UserProfile.jsx` - User profile component

## Usage

Components should be imported and used in pages or other components as needed.

Example:
```jsx
import Navbar from '../components/Navbar';
```
