# Frontend Pages

This directory contains the main page components for the TheraAI application.

## Structure

- `Home.jsx` - Landing page
- `Dashboard.jsx` - User dashboard
- `Login.jsx` - User login page
- `Register.jsx` - User registration page
- `TherapySession.jsx` - Active therapy session page
- `Profile.jsx` - User profile page
- `About.jsx` - About page

## Usage

Pages are used as route components in the main App.jsx file.

Example:
```jsx
import Home from './pages/Home';
```
